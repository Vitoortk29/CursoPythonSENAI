ATALHOS VSCODE
Ctrl + J = Abrir Terminal
Ctrl + K + C = Comentar
Ctrl + K + U = Descomentar
Ctrl + B = Minimiza a aba de pastas
Ctrl + P = Permite selecionar qual aba você quer abrir 
Ctrl + W = Fecha a aba que você está
Ctrl + <Números> = Navega entre as abas
Ctrl + D = Primeiro selecione a palavra que deseja trocar e vem apertando o comando e ele vem selecionando as outras palavras iguais
